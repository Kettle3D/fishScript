Using fishScript for installers

One of fishScript's specialities is making simple installer scripts.
Here's a guide to help you when you're developing with fishScript.

The download command

The download command is great for installers. Here's an example that uses the download command:

download kettle3d.github.io/fishScript/win.exe to C:\Program Files\fishScript\fS.exe

This would download fishScript onto the user's computer. However, you'd need to run this in administrator to get access to the Program Files folder, and it's sort of pointless seeing as the user needs fishScript to run it.
